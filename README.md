# 🎬 Media Catalog

Media Catalog — это консольное приложение на Python для каталогизации личной медиатеки. Проект позволяет хранить информацию о фильмах, книгах, играх и видеофайлах, оценивать их и собирать в структурированную базу данных.

## 📖 О проекте

Media Catalog решает проблему хаотичного хранения информации о просмотренных/прочитанных произведениях. Вместо заметок в блокноте вы получаете структурированную базу с возможностью:

- 📇 Хранения единой картотеки (название, год, жанр, тип)
- ✅ Отслеживания статуса ("Просмотрено" / "В планах")
- ⭐ Добавления оценок (1-10)
- ⬆️ Импорта данных из CSV файлов
- ⬇️ Экспорта данных в CSV
- 🎥 Добавления и воспроизведения видеофайлов

## 📄 Формат CSV файла

### Пример test.csv
CSV


title,year,genre,type,status,rating
Матрица,1999,Sci-Fi,фильм,Просмотрено,10
1984,1949,Антиутопия,книга,В планах,
Интерстеллар,2014,Sci-Fi,фильм,В планах,9

### 📌 Поля:
- title — название (обязательно)
- year — год (число от 1800 до 2100)
- genre — жанр (текст)
- type — тип: фильм, книга, игра или видео
- status — статус: Просмотрено или В планах
- rating — оценка: 1-10 (можно оставить пустым)
