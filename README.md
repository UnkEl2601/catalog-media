# :movie_camera: Media Catalog

Media Catalog — это проект на Python для каталогизации личной медиатеки. Проект позволяет хранить информацию о фильмах, книгах и играх, оценивать их и собирать в тематические подборки.

## :book: О проекте

Media Catalog решает проблему хаотичного хранения информации о просмотренных/прочитанных произведениях. Вместо заметок в блокноте или Excel вы получаете структурированную базу с возможностью:

- :card_index: Хранения единой картотеки (название, год, жанр, тип)
- :white_check_mark: Отслеживания статуса ("Просмотрено" / "В планах")
- :star: Добавления оценок и рецензий
- :dart: Группировки произведений в коллекции (подборки)
- :arrow_up: Импорта данных из CSV файлов
- :game_die: Получения рекомендаций на основе любимых жанров
